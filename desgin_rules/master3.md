------------------------------------------------------------

# CODING STANDARDS

Use semantic HTML.

Keep components small.

Avoid unnecessary nesting.

Use reusable utilities.

Never hardcode repeated values.

Use constants where appropriate.

Write clean and readable code.

Prioritize maintainability.

------------------------------------------------------------

# CSS GUIDELINES

No Tailwind.

No Bootstrap.

Only CSS.

Use CSS Modules for components.

Use global.css only for:

Typography

Variables

Utilities

Reset

Theme

Scrollbar

Selection

Container

Spacing

Everything else remains component-scoped.

------------------------------------------------------------

# RESPONSIVE STRATEGY

Desktop First

↓

Laptop

↓

Tablet

↓

Mobile

↓

Small Mobile

Never hide content.

Instead:

Reorganize

Resize

Reflow

Maintain hierarchy.

Animations should gracefully simplify on smaller devices.

------------------------------------------------------------

# GSAP RULES

GSAP controls storytelling.

Use:

ScrollTrigger

Timeline

SplitText

ClipPath

Pinning

Parallax

Reveal

Scrub only where beneficial.

Avoid excessive motion.

------------------------------------------------------------

# FRAMER MOTION RULES

Use Framer only for:

Hover

Cards

Buttons

Dropdowns

Page transitions

Micro interactions

Never use Framer for long storytelling sequences.

------------------------------------------------------------

# LENIS

Required globally.

Integrate with requestAnimationFrame.

Ensure compatibility with ScrollTrigger.

No scroll lag.

------------------------------------------------------------

# PERFORMANCE RULES

Lazy load images.

Lazy load videos.

Dynamic imports.

Code splitting.

Tree shaking.

Optimize bundle size.

Preload critical assets.

Compress images.

Use Next.js Image component.

Target:

95+ Lighthouse

60 FPS animations

Fast First Contentful Paint
