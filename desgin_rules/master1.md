# PROJECT MISSION

This is NOT a redesign.

This is a complete rethinking of the JaxiCloud experience.

The existing website already contains valuable content and branding, but the experience feels outdated, static and corporate.

Our mission is to preserve the existing brand identity while rebuilding the complete digital experience into a premium, modern, technology-first platform.

The visitor should immediately feel:

• Professional
• Innovative
• Trustworthy
• Intelligent
• Premium
• Modern
• Fast
• Reliable

The website must communicate that JaxiCloud is not simply selling fleet products.

JaxiCloud powers intelligent mobility.

------------------------------------------------------------

# CORE OBJECTIVES

The website should accomplish five primary goals.

1.
Increase trust immediately.

2.
Clearly explain complex fleet technology in a simple way.

3.
Guide visitors through a cinematic journey instead of traditional scrolling.

4.
Present products as premium technology solutions.

5.
Generate more enquiries and platform users.

------------------------------------------------------------

# AI IMPLEMENTATION RULES

The AI must NEVER immediately begin coding.

For every section it must:

1.
Analyze the section.

2.
Explain its purpose.

3.
Describe the proposed layout.

4.
Describe the storytelling.

5.
Describe animations.

6.
Suggest improvements.

7.
Wait for approval.

8.
Only after approval begin implementation.

No section should be implemented without approval.
